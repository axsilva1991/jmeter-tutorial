# jmeter-tutorial
### Como calcular a quantidade de usuários virtuais
VU = RPS * TEF
por exemplo RPS = 40 TPS
TEF = 0.5 seg
Legenda:
TEF = Tempo estimado de resposta
VU = Virtual User
RPS = Requisições por segundo
